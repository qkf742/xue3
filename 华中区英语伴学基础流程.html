鼎伴学华中区英语伴学基础流程

VIP英语教练全流程工作概况

课前准备 (T-6h～T-0)

1. 进群&打招呼

  • • 修改群备注: 英语教练-XX教练
  • • 与家长确认固定上课时间
  • • 关闭群聊可加好友

2. 账号检查

  • • 登录小程序检查词库和学生名字
  • • 登录校区账号
  • • 登录学生个人账号
  • • 确认能正常进入后台

3. 设备与环境自检

  • • 设置虚拟背景
  • • 检查网络和电源

4. 课前准备

  • • 上课前6小时发送腾讯会议链接
  • • 重命名为"学生姓名+鼎伴学VIP英语课"
  • • 会议号示例：154-827-659

上课中 (T=0)

1. 准时上课

  • • 提前5-10分钟入会
  • • 5分钟后未见学生: 语音通话
  • • 仍找不到: 联系组长/宋老师

2. 课堂结构

  • • 先复习上一轮单词
  • • 再学新词(30-45个/轮)
  • • 1小时课新词≥25个（视学生具体情况变动）

3. 教学规范

  • • 不解释单词
  • • 只做发音、拼写指导
  • • 学后检测提交后才可选新词

4. 下课前准备

  • • 打印当日单词表
  • • 保存PDF/截图

课后30分钟内

1. 学生群反馈

  • • 今日单词表PDF/图片
  • • 课堂反馈模板
  • • 21天抗遗忘复习参考
  • • 复习提醒设置

2. 个人组群登记

  • • 填写《课时登记表》
  • • 更新《抗遗忘复习表》

持续跟进 (当天～第21天)

1. 每日提醒 & 打卡

  • • 按表格日期提醒
  • • 24小时内纠音反馈

2. 阶段性检测

  • • 每10课时测试一次
  • • 结课时测试
  • • 结果PDF发给组长

3. 课程临近结束

  • • 提前2-3课时提醒
  • • 确认是否续费

注意事项速查表

好友申请限制

不能通过任何鼎伴学相关人员的微信好友申请(可加的只有学管宋元钦和自己的组长)

迟到/旷课处理

第一次谈话→第二次按正常时间上下课并记录

异常情况处理

任何异常(账号、网络、家长投诉)→先手机联系组长或宋老师
